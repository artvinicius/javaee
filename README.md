# Java Web (Servlet - jsp - JDBC

### Java WEB - Agenda de contatos

> Nesse projeto foi desenvolvido WEB back-end ou full-stack utilizado a Linguagem Java, para desenvolver uma pequena agenda de contatos onde será possível criar **CREATE**, ler **READ** , atualizar **UPDATE** e excluir **DELETE** contatos, além de gerar um relatório no formato .PDF com todos os contatos inseridos na lista. 

Tecnologias utilizadas:

+ Servidor Tomcat v9.0.62
+ Java EE (J2EE) - Servlet e JSP
+ IDE Eclipse  v2020-06
+ JDBC (Java Database Connectivity)
+ MySQL CRUD
+ MVC (Model View Controller)
+ JavaBeans
+ DAO
+ iText
